C++ · CMPSCI-235

Sorting & statistics final

A C++ program that reads exam scores from a file, computes the average, counts scores above and below it, sorts them with a selection sort, and writes a full report to a new file — combining file I/O, arrays, loops, and a classic algorithm into one complete solution.

void selectionSort(double arr[], int n){
  for(int i=0;i<n-1;i++)
    for(int j=0;j<n-i-1;j++)
      if(arr[j]>arr[j+1]){
        double t=arr[j]; arr[j]=arr[j+1]; arr[j+1]=t;
      }
}
Java · CMPSCI-111

Student registration system

An object-oriented Java program with separate Course and Student classes. A driver reads course and student data from text files, links students to courses by ID, and prints a complete roster — reinforcing classes, collections, and file I/O.

Assembly · CMPSCI-122

Architecture & assembly language

Explored how processors actually execute instructions — registers, memory, branching, and loops — by writing programs in Pep/9 assembly. This is where the abstraction fell away and I saw what "high-level code" compiles down to.
