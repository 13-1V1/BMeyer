# Brennan Meyer — Portfolio

A personal portfolio site showcasing my coursework, projects, and self-built apps
across software, electronics, engineering, welding, manufacturing, and the creative arts.

Live site: https://13-1v1.github.io/BMeyer/

## Structure

- `index.html` — home; an "areas of study" grid that zooms into each room
- Course rooms: `computer-science`, `electronics-robotics`, `engineering`, `welding`,
  `manufacturing`, `network`, `cybersecurity`, `automotive`, `creative-electives`
- `vibe-coding.html` — self-built apps & games (Snake, G-Code Trainer, BlueCAD)
- `downloads.html` — downloadable desktop apps and résumé
- `transcripts.html`, `about.html`, `contact.html`
- `assets/` — shared `css/site.css`, `js/site.js`, `images/`, `audio/`
- `demos/` — playable class programs (word games) and machinist calculators
- `vibe/` — in-browser rebuilds of my desktop apps, plus hosted BlueCAD

## Tech

Plain HTML, CSS, and vanilla JavaScript — no build step. Page transitions use a small
custom zoom effect with a reduced-motion fallback. Deployed with GitHub Pages.
