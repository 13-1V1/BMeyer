Electronics & Robotics

Where hardware meets software. From building analog circuits and reading Ohm's law to programming industrial robots and IoT devices, these courses taught me how physical systems are sensed, controlled, and automated.

ESYST-101 · Basic Electronics

Circuits from the ground up

Studied resistors, capacitors, diodes, transistors, and ICs; built and analyzed series and parallel circuits; and practiced soldering. Calculated resistor values with Ohm's law and built a blinking-LED circuit using a 555 timer.

ESYST-112 · Industrial Robotics

Vision → decision → action robot

For the final project I built a block-classification and LED-feedback system: a webcam and a Teachable-Machine model identified green and blue blocks in real time, sent Web Serial commands to an Arduino, and lit matching LEDs — a hands-on demo of the vision-decision-action pipeline used in industrial automation.

ESYST-131 · IoT / Python

Programming in Python

Though labeled an IoT class, this was a solid intro to Python — variables, control flow, user-defined functions, and object-oriented programming. Coming from C and Java, it was a chance to adopt Python's indentation-based style while applying familiar concepts.
